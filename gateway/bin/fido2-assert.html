Gateway FIDO2 Assertion for Arm Rehearsal / Confirm

RP: gateway.local. Produces the same JSON shape as fido2-helper.py assert so it can be fed to bin/arm --fido2-rehearse (or a manual confirm) on macOS where direct HID get_assertion is blocked.

Use the exact challenge (hex) from the stage and the credential_id (hex) of your enrolled fido2-es256 entry.

Assertion JSON (copy and use with FIDO2_ASSERT_JSON)



  
Usage

  
FIDO2_ASSERT_JSON='...paste above...' bin/arm --fido2-rehearse

  
Or pipe / set in your manual confirm flow.
